# TokeniCode

> 你的终端，你的风格。

个人 Shell 环境配置集 — 包含 Zsh/Bash 配置、别名、主题和效率工具链。

## 快速开始

```bash
git clone https://github.com/s3467199421-stack/tokenicode.git ~/.tokenicode
cd ~/.tokenicode
chmod +x install.sh && ./install.sh
```

## 目录结构

```
tokenicode/
├── zsh/           # Zsh 配置
├── bash/          # Bash 配置  
├── aliases/       # 通用别名
├── themes/        # 主题文件
├── scripts/       # 工具脚本
└── install.sh     # 一键安装
```

## 特性

- 智能别名管理
- 模块化配置，按需加载
- 一键安装，开箱即用
